Only 64b platforms.

MSB:
0 - GC-visible pointer
1 - immediate value

Lower 3 bits are pointer tag:
000 - normal object
001 - ImmutableList
010 - Pair
011 - Symbol
100 - Closure
101 - Primitive
110 - Macro
111 - SpecialForm

Upper byte of immediate objects:
0x80 - Nil
0x81 - Boolean
0x82 - FixNum
0x83 - Character
0x84 - UnsignedByte
0x85 - UnsignedInt16
0x86 - UnsignedInt32
0x87 - Float32
0x88 - SignedByte
0x89 - SignedInt16
0x8a - SignedInt32
0x8b - SmallFraction
0x90-0x97 - ShortString

0xbf - invalid object with flags (ie. ImmutableList header/footer)
0xc0-0xff - shifted FloNum


SmallFraction:
- 31 bits signed numerator (two's complement)
- 24 bits denominator

Boolean:
0 - false
1 - true

Nil:
0 - nil


reserved values:
0xffff ffff ffff ffff
0x0000 0000 0000 0000 (ie. C NULL)
